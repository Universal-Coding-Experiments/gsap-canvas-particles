## ğŸš€ Key Features

- **Smooth GSAP Timelines**: Leverages GSAP's powerful engine for fluid, high-frame-rate particle animations.
- **Canvas Rendering**: Uses the HTML5 Canvas API for efficient drawing, capable of handling hundreds of particles with ease.
- **Interactive UI**: Pointer events allow users to **toggle Play/Pause** by clicking anywhere on the screen.
- **Micro-Interactions**: Dynamic scaling, rotation, and easing effects that create a cinematic feel.
- **Fully Responsive**: Intelligent window resize handling ensures the canvas always fills the viewport perfectly.
- **Customizable Assets**: Easily swap particle images to create entirely different visual themes.
